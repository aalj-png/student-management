# Student Management (Minimal)

Simple Node.js full-stack demo using Express and SQLite.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Start server

```bash
npm start
# or for development: npm run dev
```

3. Open http://localhost:3000

Files added:
- server.js - Express server and API
- db.js - SQLite initialization (file: students.db)
- public/ - frontend (index.html, styles.css, main.js)
